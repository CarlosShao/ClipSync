## 文件预览功能实现计划

### 目标
让剪贴板中所有文件类型的数据都能预览内容（.md/.txt/.json/.js 等），类似编辑器的 Markdown 预览模式，带目录跳转。

---

### 第一步：安装依赖
- `marked` — Markdown → HTML 渲染
- `highlight.js` — 代码语法高亮（190+ 语言）
- Tauri 桌面端无 XSS 风险，不装 DOMPurify

### 第二步：后端 — 新增文件内容获取接口
- **新路由**: `GET /api/clipboard/:id/content`
- 返回 `content_encrypted` 字段（即实际文件内容）
- 修改文件: `src/server/src/routes/clipboard.js`

### 第三步：前端 — ModalManager 预览增强
- **文件类型检测**: 根据扩展名判断 → Markdown / 代码 / 纯文本 / 不支持
- **Markdown 渲染**: 用 `marked` 替换手写 `renderMarkdown`，为标题添加锚点 id
- **TOC 目录**: 自动提取 h1-h6 生成侧边栏目录，点击跳转
- **代码高亮**: `highlight.js` 根据扩展名自动检测语言，跟随深色/浅色主题
- **预览触发**: 点击文件预览按钮 → 调用 API 获取内容 → 选择渲染方式 → 显示模态框

### 第四步：CSS 样式
- Markdown 样式（标题/列表/代码块/表格/引用/链接）
- TOC 固定宽度侧边栏
- highlight.js 主题样式

### 预览模态框 UI
```
┌─────────────────────────────────────────────┐
│  [文件名]  [Markdown]  [1.2 KB]      [×]   │
├──────────┬──────────────────────────────────┤
│ 目录     │  渲染内容区                        │
│ ├ 标题1  │  # 标题1                          │
│ ├ 标题2  │  正文内容...                       │
│ └ 标题3  │  ## 标题2 ...                     │
└──────────┴──────────────────────────────────┘
```

### 本次不含
- Word/Excel/PDF 预览（后续单独做）
- 大文件分页 / 文件编辑
