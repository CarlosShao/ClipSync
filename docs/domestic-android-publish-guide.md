## 上架前必须准备的资质

### 1. 软件著作权（软著）⚠️ 必须

**用途**: 证明您是软件的合法著作权人，所有国内商店都要求。

| 项目 | 说明 |
|------|------|
| 申请机构 | 中国版权保护中心（官网：ccopyright.com.cn） |
| 正常周期 | 40-60 个工作日（约 2-3 个月） |
| 加急服务 | 部分代理机构可 10-20 个工作日出证（需额外费用） |
| 临时方案 | 部分商店接受"电子版权证书"（权威性较低，不推荐） |

**软著申请所需材料**:
- 软件著作权登记申请表
- 软件说明书（功能介绍 + 操作手册）
- 源代码（前 30 页 + 后 30 页，每页不少于 50 行）
- 申请人身份证明（个人：身份证；企业：营业执照）

### 2. APP 备案 ⚠️ 必须（2023年9月起强制）

**用途**: 中国工信部要求，所有在中国大陆上架的 APP 必须备案。

| 项目 | 说明 |
|------|------|
| 申请入口 | [工业互联网平台 APP 备案系统](https://beian.miit.gov.cn/) |
| 办理周期 | 3-7 个工作日 |
| 所需材料 | 营业执照（企业）或身份证（个人）+ 域名证书 + 服务器托管商信息 |
| 注意 | 个人开发者备案可能受限，建议以企业主体备案 |

### 3. ICP 备案 / ICP 许可证

| 类型 | 适用场景 | 说明 |
|------|----------|------|
| **ICP 备案** | 所有联网 APP | 服务器在国内必须备案，与 APP 备案是两个独立流程 |
| **ICP 许可证** | 经营性 APP（收费） | 需要，ClipSync 有订阅收费，需要办理 |

**ICP 许可证办理**:
- 办理周期：30-60 个工作日
- 要求：企业注册资本 ≥ 100 万（部分省份）
- 办理机构：各省通信管理局

### 4. 隐私政策 + 用户协议 ⚠️ 必须

**用途**: 所有商店都要求 APP 提供隐私政策和用户协议。

ClipSync 已有：
- ✅ `public/terms.html` - 服务条款
- ✅ `public/privacy.html` - 隐私政策
- ✅ Cookie 政策横幅

**注意**: 隐私政策需要包含：
- 收集的信息类型（手机号、邮箱、剪贴板内容）
- 信息使用目的（同步功能、推送通知）
- 信息存储位置（PostgreSQL 数据库）
- 用户权利（GDPR 数据导出、账户删除）
- 第三方共享（支付渠道、推送服务）

### 5. 应用签名证书

**用途**: 证明 APP 是您发布的，防止被篡改。

```bash
# 使用 keytool 生成签名证书
keytool -genkey -v -keystore clipsync-keystore.jks -alias clipsync -keyalg RSA -keysize 2048 -validity 10000
```

**注意**: 每个商店可以上传不同的 APK（用同一个签名证书），但华为要求**上传签名证书指纹**到开发者后台。

## 常见问题

### Q1: 个人开发者可以上架吗？

**A**: 可以，但有限制：
- 部分商店（OPPO、vivo）优先审核企业开发者
- ICP 许可证必须企业主体才能办理
- 支付渠道（微信、支付宝）需要企业商户号

**建议**: 如果计划商业化，建议注册个体工商户或有限公司。

### Q2: 软著可以加急吗？

**A**: 可以，有两种方式：
1. 自己申请时选择"加急"（不一定批准）
2. 委托代理机构（费用约 600-1500 元，10-20 工作日出证）

### Q3: APP 备案和 ICP 备案有什么区别？

| 项目 | APP 备案 | ICP 备案 |
|------|----------|----------|
| 主管机构 | 工信部 | 工信部 |
| 适用对象 | APP 本身 | 服务器域名 |
| 办理顺序 | 先 ICP 备案，再 APP 备案 | 先办这个 |
| 是否需要 | 必须 | 必须 |

### Q4: 可以在多个商店同时上架吗？

**A**: 可以，没有任何限制。甚至可以每个商店用不同的包名（但不建议）。
