## 五、剩余优化项（按优先级）

### 高优先级（P0 - 生产环境必须）
1. **PostgreSQL HA 部署** - 主从复制 + 自动故障转移
2. **Redis Sentinel 部署** - 高可用 Redis
3. **外部服务集成测试** - Stripe、短信服务集成时应用断路器

### 中优先级（P1 - 生产环境推荐）
1. **消息队列** - Bull 或 Redis Streams（邮件、支付回调）
2. **Node.js 集群模式** - 利用多核 CPU
3. **WebSocket 全局连接限制** - 防止资源耗尽
4. **Grafana 仪表盘** - 可视化监控指标
5. **告警规则测试** - 验证 Prometheus Alertmanager 告警

### 低优先级（P2 - 可延后）
1. **API 版本控制** - `/api/v1/`
2. **浏览器缓存** - 静态资源缓存头
3. **配置热更新** - 无需重启服务
4. **OpenAPI 文档** - 自动生成 API 文档
